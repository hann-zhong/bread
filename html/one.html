strings and arrays

some useful notes

to access individual characters, index with positive and negative integers.

str = "hello"
str[0] => h
str[-1] => o
//to return part of a string: str[x:y], not inclusive
str[1:3] => el


we can delimit strings with the split() function. setting the maxsplit param will return a list of maxsplit + 1 elements

split(regex, maxsplit)
//default regex is whitespace
str = "new york!seattle!los angeles!chicago!boston"
str.split("!") => ['new york', 'seattle', 'los angeles', 'chicago', 'boston']
str.split("!", 2) => ['new york', 'seattle', 'los angeles!chicago!boston']


string and array questions can be handled with a similar approach most of the time.
