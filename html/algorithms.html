binary search

useful to search for an element in a sorted array. binary search works by repeateadly halving the search interval; therefore, time complexity = O(log n).

def binarySearch(arr, target):
 left = 0
 right = len(arr) - 1

 while left <= right:
  mid = (left + right) / 2
  if arr[mid] == target:
   return mid
  elif arr[mid] < target:
   left = mid + 1
  elif arr[mid] > target:
   right = mid - 1
